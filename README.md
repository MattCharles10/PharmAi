## ✨ Features

### 🔍 Medicine Recognition
- **AI-Powered Scanning** - Identify medicines using TensorFlow Lite ML models
- **Barcode Scanner** - Scan medicine barcodes instantly
- **Expiry Date Detection** - Automatically detect expiry dates from packaging
- **Label Reading** - OCR to read medicine labels

### 📦 Inventory Management
- **Medicine Inventory** - Track all your medicines in one place
- **Expiry Alerts** - Get notified before medicines expire
- **Batch Tracking** - Track batch numbers and purchase dates
- **Barcode Integration** - Quick add via barcode scanning

### ⏰ Smart Reminders
- **Medicine Reminders** - Never miss a dose
- **Custom Schedules** - Daily, weekly, or custom frequency
- **Notifications** - Push notifications for reminders
- **History Tracking** - Track taken/missed doses

### 🔬 Drug Information
- **FDA Database** - Search FDA drug database
- **Drug Details** - Indications, side effects, warnings
- **Favorites** - Save frequently searched medicines
- **Alternatives** - Find drug alternatives and generics

### 👤 User Features
- **Biometric Authentication** - Secure login with fingerprint/face
- **Dark Mode** - Eye-friendly dark theme
- **Data Export** - Export your medicine data
- **Multi-language Ready** - Prepared for localization
